BookFinder Details Site

Product Description

Our project is a system for buying and selling of books in a method cost efficient to both the buyer and the seller. It allows each individual to cross-reference and compare book prices between online retailers and other students. This allows both the buyer and the seller to make an informed decision about pricing and purchasing of books.

  • Price comparisons across multiple retailers
  • User accounts with user data stored
